O Plugin Localização WP fornece informações geográficas detectadas por um endereço IP além de simular o valor do frete dos correios na página de produtos.

== Description ==

* PORTUGUESE *
O Plugin Localização WP fornece informações geográficas detectadas por um endereço IP. Isso pode ser usado em temas ou outros plug-ins como um shortcode.

* ENGLISH *
Plugin Localização WP provides geographic information detected by an IP address. This can be used in themes or other plugins as a shortcode.

* ESPAÑOL *
El plugin Plugin Localização WP proporciona información geográfica detectada por una dirección IP. Esto se puede usar en temas u otros complementos como un shortcode.

Utilizamos os serviços tercerizados da API https://ip-api.com/, cujo termos de uso são encontrados no link https://ip-api.com/docs/legal

== Installation ==

= For an automatic installation through WordPress: =

1. Select **Add New** from the WordPress **Plugins** menu in the admin area.
2. Search for **Localização WP**.
3. Click **Install Now**, then **Activate Plugin**.

= For manual installation via FTP: =

1. Upload the **localizacao-wp** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin from the **Plugins** screen in your WordPress admin area.

= To upload the plugin through WordPress, instead of FTP: =

1. From the **Add New** plugins page in your WordPress admin area, select the **Upload Plugin** button.
2. Select the **localização-wp.zip** file, click **Install Now** and **Activate Plugin**.
